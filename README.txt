# Static Spreadsheet Viewer

This site was generated from **Baseball.xlsx**.

## How to publish
1. Download `baseball_site.zip` from the link I provide.
2. Unzip it.
3. Create a new GitHub repository (or use an existing one).
4. Commit the files and push to GitHub.
5. Enable **GitHub Pages** in the repository settings, with source set to `main` branch (root).
6. Your site will be live at the URL GitHub provides.

You can also drag-and-drop the folder to **Netlify** or upload to any static web host.
